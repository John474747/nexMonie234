# **App Name**: Nex Monie

## Core Features:

- Financial Command Center: A high-fidelity dashboard displaying Available Balance (₦1,250,000.00) with a privacy eye toggle, decorative wave graphic overlays, and a quick-action row for Fund, Send, Receive, and Account Details.
- NexTips Advisor Tool: An AI-powered banner tool that suggests personalized 'NexTips' such as automated daily savings and consistent wealth growth strategies based on user spending patterns.
- Quick Action Grid: An interactive 4x2 grid of 8 square cards for rapid access to core services like Pay Bills, Buy Airtime, Data Bundles, and Refer & Earn.
- Product Discovery Hub: A horizontal scrolling row of specialized financial product cards including NexWallet, high-yield Savings (18% interest), Investments, and Insurance.
- Growth Snapshot: A financial analytics module showing real-time monthly trends for Total Inflow, Total Outflow, and Net Growth with percentage indicators and directional status icons.
- Premium Navigation Central: A fixed bottom navigation bar featuring an oversized teal #005F56 floating action button with a plus icon for core transaction workflows.

## Style Guidelines:

- Primary: Deep Mint (#005F56) for balance cards; Accent: Forest Sage (#1B816B); Highlight: Coral (#FF6B6B) for iconography; Background: Pearl White (#F8FAF9).
- Primary font: 'Poppins' throughout. Heading: 22px bold; Balance: 28px bold; Section Titles: 16px bold; Labels: 11px-12px grey text.
- Material Design 3 outlined icons; Quick actions use coral pink icons in light background circles; Top bar features a teal 'N' logo.
- Modular card design with 24px border radius for major cards and 16px for smaller tiles; consistent 16px horizontal padding throughout.
- Subtle scale transformations on action tiles and micro-interactions on the central floating action button to reinforce the premium experience.
